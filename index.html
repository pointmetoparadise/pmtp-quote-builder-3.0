Setup Guide one-time setup
Do these steps once. After that, just fill in the builder and copy the prompt for every new quote.
Part 1 — Web3Forms (emails you every time a client submits)
1
Go to web3forms.com. Click "Get Started Free."
2
Enter your work email and click "Get your Access Key."
Use the email where you want to receive client selections.
3
Check your inbox for a Web3Forms confirmation email. Click the confirmation link.
Check spam if you don't see it in 2 minutes.
4
Copy your Access Key (looks like 7b40d437-1004-48e2-9031-171a8b0ef4f1) and paste it into Agent Info above.
5
Log in at app.web3forms.com anytime to see all submissions.
Done. One key works for all quotes forever.
Part 2 — GitHub (hosts your live quote pages for free)
1
Go to github.com. Click "Sign up" and create a free account.
2
Click "+" top right → "New repository."
3
Name it pmtp-quotes. Select "Public." Click "Create repository."
4
Click "Settings" → "Pages" in the left sidebar.
5
Under Branch select "main" → folder "/ (root)" → click "Save."
6
Wait 2–3 min. Green banner appears with your live URL. Paste it into Agent Info above.
Done. Upload all future quotes to this same repo.
Part 3 — Uploading a Quote (every new quote)
1
Download the HTML file Claude built.
2
Go to your repo on GitHub → click "Add file" → "Upload files."
3
Drag your file in → click "Commit changes."
4
Wait 2–3 min. Live at: https://[username].github.io/pmtp-quotes/[filename].html
Replacing a file: Upload same filename → wait 3 min → add ?v=2 to URL to force reload.
404 error? Wait 3 more minutes. Filename in URL must exactly match — capital letters matter.
All quotes in one repo. Never create another repo.
Part 4 — Test Before Sending
1
Open your live quote URL in a browser.
2
Select options — running total should update instantly.
3
Check/uncheck add-ons — total should update live.
4
Submit with your own name/email. Web3Forms email should arrive in 1–2 min.
5
All good? Send the client their PDF — last page links to the selection page.
